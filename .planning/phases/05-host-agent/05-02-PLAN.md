    **agent/locus_agent/terminal/unix.py** (per D-21, D-23, D-24):
    - `class UnixSessionManager`:
    - `TMUX_PREFIX = "locus-"`
    - `async def create_session(self, session_id: str, cols: int, rows: int, working_dir: str | None = None, tmux_session: str | None = None) -> dict`:
      - If tmux_session provided: `tmux attach -t '{tmux_session}'`
      - Else: generate name `{TMUX_PREFIX}{session_id[:8]}`, command `tmux new-session -s '{name}'` with `-c '{working_dir}'` if provided
      - Use `pty.openpty()` + `os.fork()` pattern from LocalTerminalProcess
      - In child: `os.setsid()`, dup2 slave to 0/1/2, close fds, set TERM=xterm-256color, `os.execvp("bash", ["bash", "-c", cmd])`
      - In parent: close slave, set terminal size via `fcntl.ioctl(master_fd, termios.TIOCSWINSZ, struct.pack("HHHH", rows, cols, 0, 0))`
      - Make master_fd non-blocking: `fcntl.fcntl(master_fd, fcntl.F_SETFL, flags | os.O_NONBLOCK)`
      - Return `{"master_fd": master_fd, "pid": pid, "tmux_name": name}`
    - `async def read(self, session_data: dict, n: int = 16384) -> bytes`: async read from master_fd using event loop add_reader
    - `def write(self, session_data: dict, data: bytes)`: `os.write(master_fd, data)`
    - `def resize(self, session_data: dict, cols: int, rows: int)`: ioctl TIOCSWINSZ
    - `def close(self, session_data: dict)`: Remove reader, close master_fd, waitpid WNOHANG
    - `async def list_sessions(self) -> list[dict]`: Run `tmux ls -F '#{session_name}:#{session_attached}:#{session_activity}'`, filter locus- prefixed, return list of dicts
    - `async def rediscover(self) -> list[str]`: List tmux sessions matching locus-* prefix (for agent restart recovery per D-24)
    - NOTE: No agent-side scrollback on Unix — tmux handles it (per D-23)

      - `_scrollback: dict[str, bytearray]` — only used on Windows; on Unix this stays empty
      - `_read_tasks: dict[str, asyncio.Task]` — background read loops per session
      - `MAX_SCROLLBACK = 65536`
      - `async def create(self, session_id: str, cols: int, rows: int, working_dir: str | None = None, tmux_session: str | None = None) -> dict`:

      - `async def _read_loop(self, session_id: str)`:
        Loop reading from manager, buffer in scrollback (Windows only — on Unix skip buffering), forward to attached WS if any

      - `def detach(self, session_id: str)`: Set _ws_map[session_id] = None

      - `def write(self, session_id: str, data: bytes)`: Forward to manager.write
      - `def resize(self, session_id: str, cols: int, rows: int)`: Forward to manager.resize
      - `async def close(self, session_id: str)`: Cancel read task, close via manager, remove from dicts
      - `async def close_all(self)`: Close all sessions (for shutdown)
      - `def list_active(self) -> list[str]`: Return list of active session IDs
