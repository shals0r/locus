    **backend/app/agent/deployer.py** (per D-12, D-13, D-14, D-15, D-16, D-17):

    Import: asyncssh, httpx, logging, os, pathlib

    Constants:
    - `AGENT_DIR = "~/.locus-agent"`
    - `AGENT_PORT = 7700` (default, configurable)
    - `AGENT_PYZ_DIR = "/app/agents"` (inside Docker image — bundled .pyz files per D-16)

    **Platform detection:**
    - `async def detect_platform(ssh_conn) -> str`:
      - Run `uname -s 2>/dev/null || echo Windows` via ssh_conn.run()
      - Map: "Linux" -> "linux-x64", "Darwin" -> "darwin-arm64", "Windows" -> "win-x64"
      - For Darwin, also check `uname -m` to distinguish arm64 vs x86_64

    **Python detection** (per Research Open Question 2):
    - `async def detect_python(ssh_conn) -> str | None`:
      - Try in order: `python3 --version`, `python --version`, `py -3 --version`
      - Return the command name that responds with "Python 3.12" or higher
      - Return None if no suitable Python found

    **Agent probing:**
    - `async def probe_agent(base_url: str, token: str | None = None) -> dict | None`:
      - Try httpx.AsyncClient GET to `{base_url}/health` with 3s timeout
      - If responds, return health JSON. If connection refused/timeout, return None.
      - If token provided, also verify auth works via a simple authenticated request.

    **Deployment:**
    - `async def deploy_agent(ssh_conn, platform: str | None = None, port: int = AGENT_PORT) -> tuple[str, str]`:
      - Returns (base_url, token)
      - Step 1: Detect platform if not provided
      - Step 2: Detect Python interpreter, raise RuntimeError if none found
      - Step 3: Create remote directory: `ssh_conn.run("mkdir -p ~/.locus-agent")`
      - Step 4: SCP correct .pyz file:
        ```python
        pyz_name = f"locus-agent-{platform}.pyz"
        local_path = os.path.join(AGENT_PYZ_DIR, pyz_name)
        await asyncssh.scp(local_path, (ssh_conn, f"~/.locus-agent/{pyz_name}"))
        ```
      - Step 5: Start agent in background:
        - Unix: `nohup {python_cmd} ~/.locus-agent/{pyz_name} start > ~/.locus-agent/agent.log 2>&1 &`
        - Windows: `Start-Process -NoNewWindow {python_cmd} -ArgumentList '~/.locus-agent/{pyz_name}','start'`
      - Step 6: Wait for agent to come online: poll /health every 0.5s for up to 10s
      - Step 7: Read token: `ssh_conn.run("cat ~/.locus-agent/agent.token")`
      - Step 8: Determine base_url: extract host from ssh_conn, build `http://{host}:{port}`
      - Return (base_url, token)

    **Ensure agent (probe + deploy):**
    - `async def ensure_agent(ssh_conn, host: str, port: int = AGENT_PORT, expected_version: str | None = None) -> tuple[str, str]`:
      - Probe agent at `http://{host}:{port}`
      - If running and version matches (or expected_version is None): read token, return
      - If running but version mismatch (per D-17): stop old agent via SSH `{python_cmd} ~/.locus-agent/*.pyz stop`, then deploy
      - If not running: deploy
      - Return (base_url, token)
  </action>
