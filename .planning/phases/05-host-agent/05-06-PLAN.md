    **agent/locus_agent/api/files.py** (per D-07):

    FastAPI APIRouter, prefix="", tag "files". All endpoints require `Depends(verify_token)`.

    **GET /files/read** — Read a file:
    - Query params: `path: str`
    - Read file content, return `{"path": path, "content": str, "size": int, "encoding": "utf-8"}`
    - For binary files, base64 encode and set `"encoding": "base64"`
    - Max file size: 10MB. Return 413 if exceeded.
    - Return 404 if file not found.

    **POST /files/write** — Write a file:
    - Body: `{"path": str, "content": str, "encoding": str = "utf-8"}`
    - Write content to path. Create parent dirs with `os.makedirs(parent, exist_ok=True)`.
    - Return `{"path": path, "size": int, "written": True}`

    **GET /files/list** — List directory:
    - Query params: `path: str, recursive: bool = False`
    - List directory entries: `os.scandir(path)` or `os.walk(path)` if recursive
    - Return `{"path": path, "entries": [{"name": str, "type": "file"|"dir"|"symlink", "size": int, "modified": float}]}`
    - Return 404 if path not found.

    **GET /files/stat** — Stat a file:
    - Query params: `path: str`
    - Return `{"path": path, "exists": bool, "type": "file"|"dir"|"symlink", "size": int, "modified": float, "permissions": str}`

    **POST /files/search** — Search files (ripgrep):
    - Body: `{"path": str, "pattern": str, "glob": str | None, "max_results": int = 100}`
    - Run `rg --json -m {max_results} {pattern} {path}` via subprocess
    - Parse JSON output, return `{"matches": [{"path": str, "line": int, "text": str}]}`
    - If rg not found, fall back to `grep -rn` with limited output

    **agent/locus_agent/api/git.py** (per D-08):

    FastAPI APIRouter, prefix="", tag "git". All endpoints require `Depends(verify_token)`.

    Helper: `async def _run_git(repo_path: str, *args) -> tuple[int, str, str]`: Run `git -C {repo_path} {args}` via asyncio.create_subprocess_exec. Return (returncode, stdout, stderr). Return 503 if git not found.

    **GET /git/status** — Repository status:
    - Query params: `repo_path: str`
    - Run `git -C {repo_path} status --porcelain=v2 --branch`
    - Parse output: extract branch, ahead/behind, changed files
    - Return `{"repo_path": path, "branch": str, "clean": bool, "ahead": int, "behind": int, "files": [{"path": str, "status": str}]}`

    **GET /git/branches** — List branches:
    - Query params: `repo_path: str`
    - Run `git -C {repo_path} branch -a --format='%(refname:short):%(objectname:short):%(upstream:short)'`
    - Return `{"branches": [{"name": str, "commit": str, "upstream": str | None, "current": bool}]}`

    **GET /git/log** — Commit log:
    - Query params: `repo_path: str, count: int = 20`
    - Run `git -C {repo_path} log --format='%H|%h|%an|%ae|%at|%s' -n {count}`
    - Return `{"commits": [{"hash": str, "short_hash": str, "author": str, "email": str, "timestamp": int, "message": str}]}`

    **GET /git/diff** — Get diff:
    - Query params: `repo_path: str, ref: str | None = None, cached: bool = False`
    - Run `git -C {repo_path} diff {--cached if cached} {ref if ref}`
    - Return `{"diff": str, "stats": {"files_changed": int, "insertions": int, "deletions": int}}`
    - Also run `git diff --stat` for stats.

    **POST /git/exec** — Generic git command:
    - Body: `{"repo_path": str, "args": list[str]}`
    - Run `git -C {repo_path} {args}`
    - Return `{"returncode": int, "stdout": str, "stderr": str}`
    - Blocklist dangerous commands: `["push --force", "reset --hard", "clean -f"]` — return 403

    **backend/app/agent/client.py** — Add file and git methods:

    ```python
    # File operations
    async def read_file(self, path: str) -> dict:
        resp = await self._http.get("/files/read", params={"path": path})
        resp.raise_for_status()
        return resp.json()

    async def write_file(self, path: str, content: str) -> dict:
        resp = await self._http.post("/files/write", json={"path": path, "content": content})
        resp.raise_for_status()
        return resp.json()

    async def list_directory(self, path: str, recursive: bool = False) -> dict:
        resp = await self._http.get("/files/list", params={"path": path, "recursive": recursive})
        resp.raise_for_status()
        return resp.json()

    async def stat_file(self, path: str) -> dict:
        resp = await self._http.get("/files/stat", params={"path": path})
        resp.raise_for_status()
        return resp.json()

    # Git operations
    async def git_status(self, repo_path: str) -> dict:
        resp = await self._http.get("/git/status", params={"repo_path": repo_path})
        resp.raise_for_status()
        return resp.json()

    async def git_branches(self, repo_path: str) -> dict:
        resp = await self._http.get("/git/branches", params={"repo_path": repo_path})
        resp.raise_for_status()
        return resp.json()

    async def git_log(self, repo_path: str, count: int = 20) -> dict:
        resp = await self._http.get("/git/log", params={"repo_path": repo_path, "count": count})
        resp.raise_for_status()
        return resp.json()

    async def git_diff(self, repo_path: str, ref: str | None = None, cached: bool = False) -> dict:
        params = {"repo_path": repo_path, "cached": cached}
        if ref: params["ref"] = ref
        resp = await self._http.get("/git/diff", params=params)
        resp.raise_for_status()
        return resp.json()

    async def git_exec(self, repo_path: str, args: list[str]) -> dict:
        resp = await self._http.post("/git/exec", json={"repo_path": repo_path, "args": args})
        resp.raise_for_status()
        return resp.json()
    ```
