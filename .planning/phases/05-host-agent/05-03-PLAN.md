  <action>
    Create `agent/locus_agent/api/tmux.py` (per D-10):

    - FastAPI APIRouter with prefix="" tag "tmux"
    - All endpoints require `Depends(verify_token)`
    - Helper `async def _run_tmux(*args) -> tuple[int, str, str]`: Run tmux via `asyncio.create_subprocess_exec("tmux", *args, stdout=PIPE, stderr=PIPE)`, return (returncode, stdout, stderr). Handle `FileNotFoundError` gracefully returning (1, "", "tmux not found").

    **GET /tmux/sessions** — List all tmux sessions:
    - Run `tmux ls -F '#{session_name}:#{session_attached}:#{session_activity}'`
    - Parse output into list of `{"name": str, "attached": bool, "last_activity": int}`
    - Return `{"sessions": [...], "tmux_available": True}`
    - If tmux not found or no server running, return `{"sessions": [], "tmux_available": False}`

    **POST /tmux/sessions** — Create a new tmux session:
    - Request body: `{"name": str | None, "working_dir": str | None}`
    - If name is None, generate: `locus-{os.urandom(4).hex()}`
    - Build command: `tmux new-session -d -s '{name}'` with `-c '{working_dir}'` if provided
    - Run command, return `{"name": name, "created": True}` on success
    - Return 409 if session already exists

    **GET /tmux/sessions/{name}** — Check if session exists:
    - Run `tmux has-session -t '{name}'`
    - Return `{"name": name, "exists": returncode == 0}`

    **DELETE /tmux/sessions/{name}** — Kill a tmux session:
    - Run `tmux kill-session -t '{name}'`
    - Return `{"name": name, "killed": True}` on success, 404 if not found

    **Windows handling:** All endpoints check `sys.platform == "win32"`. If Windows, return `{"sessions": [], "tmux_available": False}` for list, and 501 for create/kill with message "tmux not available on Windows".
  </action>
