    **agent/locus_agent/cli.py** (per D-14, D-19, D-02):
    - AGENT_DIR = `os.path.expanduser("~/.locus-agent")`
    - PID_FILE = `os.path.join(AGENT_DIR, "agent.pid")`
    - TOKEN_FILE = `os.path.join(AGENT_DIR, "agent.token")`
    - LOG_FILE = `os.path.join(AGENT_DIR, "agent.log")`
    - `ensure_agent_dir()`: create AGENT_DIR with mode 0o700 if not exists
    - `ensure_token() -> str`: If TOKEN_FILE exists, read and return. Otherwise generate with `secrets.token_urlsafe(32)`, write using `os.open(TOKEN_FILE, os.O_WRONLY | os.O_CREAT | os.O_TRUNC, 0o600)` (per Pitfall 6), return token.
    - `write_pid()`: Write `os.getpid()` to PID_FILE. Register `atexit.register(remove_pid)`.
    - `remove_pid()`: Remove PID_FILE if exists.
    - `read_pid() -> int | None`: Read PID_FILE, check if process is running via `os.kill(pid, 0)`, return pid or None.
    - `cmd_start(daemon: bool = False)`: Check read_pid() — if running, print "Agent already running (PID {pid})" and exit(1). Call ensure_agent_dir(), ensure_token(), set LOCUS_AGENT_TOKEN env var, call write_pid(). If daemon=True, fork and detach (Unix) or use subprocess on Windows. Import uvicorn and app.create_app(), run uvicorn with host/port from settings.
    - `cmd_stop()`: Read PID, send SIGTERM (Unix) or taskkill (Windows), wait up to 5s, print result.
    - `cmd_status()`: Read PID, if running print "Agent running (PID {pid}) on port {port}", else print "Agent not running".
    - `cmd_logs()`: Tail LOG_FILE last 50 lines if exists, else print "No logs found".

    ```bash
    #!/usr/bin/env bash
    set -euo pipefail
    VERSION=$(python -c "from locus_agent import __version__; print(__version__)")
    PLATFORM=${1:-linux-x64}
    shiv \
      --python '/usr/bin/env python3' \
      -e 'locus_agent.__main__:main' \
      -o "dist/locus-agent-${PLATFORM}.pyz" \
      --build-id "v${VERSION}" \
      --compressed \
      -r requirements.txt \
      .

    ```
