    - `async def connect(self, machine_id: str, host: str, port: int, username: str, ssh_key_path: str) -> asyncssh.SSHClientConnection`:
      Call `asyncssh.connect(host, port=port, username=username, client_keys=[ssh_key_path], known_hosts=None, keepalive_interval=15, keepalive_count_max=3)`.
      Store in `_connections[machine_id]`. Start heartbeat task. Set status "online". Return connection.

    - `async def get_connection(self, machine_id: str) -> asyncssh.SSHClientConnection | None`:
      Return `_connections.get(machine_id)`. Return None if not connected.

    - `async def disconnect(self, machine_id: str)`:
      Cancel heartbeat task. Close connection. Remove from dicts. Set status "offline".

    - `async def _heartbeat_loop(self, machine_id: str, conn: asyncssh.SSHClientConnection, host: str, port: int, username: str, ssh_key_path: str)`:
      Loop every `settings.heartbeat_interval_seconds` (30s). Run `conn.run("echo ok", check=True)` with `asyncio.wait_for` timeout 10s.
      On failure: set status "reconnecting", call `_reconnect()`.

    - `async def _reconnect(self, machine_id: str, host: str, port: int, username: str, ssh_key_path: str)`:
      Exponential backoff per D-13: initial delay `settings.reconnect_initial_delay` (1s), max delay `settings.reconnect_max_delay` (30s), max retries `settings.reconnect_max_retries` (10).
      On each attempt: try `connect()`. On success: set status "online", notify callbacks. On all retries exhausted: set status "offline", notify callbacks.

    - `def on_status_change(self, callback: Callable[[str, str], None])`: Register callback(machine_id, status).

    - `def get_status(self, machine_id: str) -> str`: Return `_machine_status.get(machine_id, "offline")`.

    - `async def shutdown(self)`: Disconnect all machines, cancel all tasks. Called from FastAPI lifespan.

    Create module-level singleton: `ssh_manager = SSHManager()`

    **backend/app/ssh/tmux.py**:
    ```python
    import asyncssh
    from uuid import uuid4

    async def list_tmux_sessions(conn: asyncssh.SSHClientConnection) -> list[dict]:
        """List tmux sessions on remote machine."""
        try:
            result = await conn.run(
                "tmux ls -F '#{session_name}:#{session_attached}:#{session_activity}'",
                check=True,
            )
            sessions = []
            for line in result.stdout.strip().split("\n"):
                if not line:
                    continue
                parts = line.split(":")
                if len(parts) >= 3:
                    sessions.append({
                        "name": parts[0],
                        "attached": int(parts[1]) > 0,
                        "last_activity": int(parts[2]),
                    })
            return sessions
        except asyncssh.ProcessError:
            return []  # no tmux server running

    async def create_terminal_in_tmux(
        conn: asyncssh.SSHClientConnection,
        session_name: str | None = None,
        working_dir: str | None = None,
        cols: int = 120,
        rows: int = 40,
    ) -> asyncssh.SSHClientProcess:
        """Create or attach to a tmux session with PTY."""
        if session_name:
            command = f"tmux attach -t {session_name}"
        else:
            name = f"locus-{uuid4().hex[:8]}"
            cd_flag = f" -c {working_dir}" if working_dir else ""
            command = f"tmux new-session -s {name}{cd_flag}"

        return await conn.create_process(
            command,
            term_type="xterm-256color",
            term_size=(cols, rows),
            encoding=None,  # raw bytes mode
        )
    ```

            encoding=None,
        )

    **backend/app/ws/terminal.py** — WebSocket endpoint:
    - Route: `@app.websocket("/ws/terminal/{session_id}")` — or use an APIRouter
    - Accept WebSocket connection
    - Authenticate: extract `token` from query params, validate JWT. Close with 4001 if invalid.
    - Look up TerminalSession by session_id from DB to get machine_id, tmux_session_name
    - Get SSH connection from `ssh_manager.get_connection(machine_id)`
    - If tmux_session_name: call `create_terminal_in_tmux(conn, session_name=tmux_session_name, cols=cols, rows=rows)`
    - Else: call `create_terminal_in_tmux(conn, working_dir=repo_path, cols=cols, rows=rows)` (always wrap in tmux per anti-pattern guidance)
    - Start two concurrent tasks (asyncio.gather):
      1. `read_from_ssh()`: Loop reading `process.stdout.read(4096)`, send as `websocket.send_bytes(data)`. On SSH error, close websocket.
      2. `read_from_ws()`: Loop receiving from websocket. If binary frame: `process.stdin.write(data)`. If text frame: parse JSON, handle `{"type": "resize", "cols": N, "rows": N}` by calling `process.channel.set_terminal_size(cols, rows)`. On WebSocketDisconnect: close process.
    - Wrap in try/finally: always call `process.close()` on exit (per Pitfall 4)
