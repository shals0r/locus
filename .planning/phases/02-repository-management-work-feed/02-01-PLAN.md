**git.py:**
- RepoStatus: branch (str), is_dirty (bool), changed_count (int), ahead (int), behind (int), last_activity (str | None)
- CommitEntry: sha (str), message (str), author (str), date (str)
- ChangedFile: status (str), path (str)
- BranchInfo: name (str), is_current (bool)
- GitOpResult: success (bool), message (str)
- RepoDetail: machine_id (str), repo_path (str), name (str), status (RepoStatus)

**feed.py:**
- IngestPayload: source_type (str), external_id (str), title (str), snippet (str | None), url (str | None), tier_hint (str | None), source_icon (str | None), metadata (dict | None)
- FeedItemResponse: all FeedItem fields as response model with id as str (UUID serialized)
- FeedItemUpdate: is_read (bool | None), is_dismissed (bool | None), snoozed_until (datetime | None), tier (str | None)
- FeedListResponse: items (list[FeedItemResponse]), total (int)

**task.py:**
- TaskCreate: title (str), context (str | None), tier (str), feed_item_id (str | None), source_links (dict | None)
- TaskResponse: all Task fields as response model
- TaskTransition: status (str), machine_id (str | None), repo_path (str | None), branch (str | None)
- TaskUpdate: title (str | None), context (str | None)

**command_palette.py:**
- SearchResult: id (str), type (str -- "repo"/"machine"/"feed_item"/"task"/"action"), title (str), subtitle (str | None), icon (str | None), action_data (dict | None)
- SearchResponse: results (list[SearchResult])
