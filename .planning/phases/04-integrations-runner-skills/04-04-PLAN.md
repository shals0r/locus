**Skill service** (`backend/app/services/skill_service.py`):

Follow research Pattern 5. Implement SSH-based skill discovery with in-memory TTL cache:

```python
import logging
from time import time

logger = logging.getLogger(__name__)

_skill_cache: dict[str, tuple[float, list[dict]]] = {}
SKILL_TTL = 300  # 5 minutes per D-22

async def discover_skills(conn, repo_path: str) -> list[dict]:
    """Discover Claude Code skills for a repo via SSH.

    Scans both .claude/commands/*.md (flat files) and
    .claude/skills/*/SKILL.md (directory structure) per D-22 and Pitfall 7.

    Args:
        conn: asyncssh connection (or None for local machine)
        repo_path: absolute path to the repository

    Returns:
        List of dicts with keys: name, description, path
    """
```

Implementation:
1. Check cache: key = `f"{id(conn)}:{repo_path}"`. If cached and within TTL, return cached value.
2. Scan `.claude/commands/*.md`:
   - Run `ls {repo_path}/.claude/commands/*.md 2>/dev/null` via SSH (or subprocess for local)
   - For each file: parse name from filename (remove `.md` extension), read first 5 lines via `head -5 {filepath}`, extract description from first non-frontmatter non-empty line (skip `---` YAML frontmatter blocks), strip leading `#` if present
3. Scan `.claude/skills/*/SKILL.md` (per research Pitfall 7):
   - Run `ls -d {repo_path}/.claude/skills/*/SKILL.md 2>/dev/null` via SSH
   - For each SKILL.md: parse name from parent directory name, read first 10 lines, look for `name:` and `description:` in YAML frontmatter, fall back to directory name and first content line
4. Combine both lists, dedup by name (commands/ takes precedence)
5. Cache results with current timestamp
6. On any SSH/subprocess error, return empty list (skills are optional, never block)

For local machine (conn=None), use `asyncio.create_subprocess_exec("ls", ...)` instead of `conn.run(...)`. Follow the existing pattern from `machine_registry.py` for local vs remote command execution.

```tsx
export function SkillBar({ machineId, repoPath }: Props) {
  const { skillsByRepo, loadingRepos, fetchSkills, activeSkills } = useSkillStore();
  const { createSession } = useSessionStore();  // or however sessions are created

  const cacheKey = `${machineId}:${repoPath}`;
  const skills = skillsByRepo[cacheKey];
  const loading = loadingRepos.has(cacheKey);

  // Fetch on mount if not cached
  useEffect(() => {
    if (skills === undefined && !loading) {
      fetchSkills(machineId, repoPath);
    }
  }, [machineId, repoPath]);

  // Don't render if no skills found (D-24)
  if (skills !== undefined && skills.length === 0) return null;

  // Loading state: single skeleton chip
  if (loading || skills === undefined) {
    return (
      <div className="mt-2">
        <div className="text-xs text-muted mb-1">Skills</div>
        <div className="h-6 w-16 rounded-md bg-secondary animate-pulse" />
      </div>
    );
  }

  return (
    <div className="mt-2">
      <div className="text-xs text-muted mb-1">Skills</div>
      <div className="flex flex-wrap gap-2">
        {skills.map((skill) => (
          <SkillChip
            key={skill.name}
            skill={skill}
            active={!!activeSkills[skill.name]}
            onClick={() => handleSkillClick(skill)}
          />
        ))}
      </div>
    </div>
  );
}
```
