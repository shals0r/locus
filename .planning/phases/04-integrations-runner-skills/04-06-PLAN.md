    **What was built:**

    Complete Phase 4: Integrations Runner and Skills system.

    **Worker Management (Settings > Integrations):**
    - Worker card list with status dots (running/stopped/crashed/disabled)
    - Play/pause buttons to start/stop workers
    - Gear icon opens quick config popover (poll interval, credential, enabled toggle)
    - Click card to expand log panel with streaming logs
    - "New Integration" button to open Integrator chat

    **Integrator Chat (side panel):**
    - Fixed right-edge resizable chat panel
    - Machine selector in header for Claude Code routing
    - Chat messages with structured cards (config steps, test results, deploy)
    - Deploy button that creates and starts a worker
    - Credential feedback: "Credentials saved securely -- Claude never sees your credentials."

    **Skills (sidebar):**
    - Skill chips appear under selected repo info when repo has .claude/commands/ files
    - Clicking a chip opens Claude Code terminal session with the skill command
    - Active chips show accent border when session is running
    - No chips shown for repos without skills

    **Verification steps for the user:**

    Worker Management:

    5. Click a worker card body to expand log panel, verify logs stream
    6. Click gear icon, verify quick config popover with poll interval and enabled toggle
    7. Click "New Integration" button, verify Integrator panel slides in from right

    Integrator Chat:
    8. Verify Integrator panel layout: header with "Integrator" title, machine selector, close button
    9. Select a machine from the dropdown (if available)
    10. Type a message in the input area, verify Enter sends it
    11. Verify assistant response appears (if Claude Code is available on a machine)
    12. Verify close button (X) and Escape key close the panel
    13. Verify panel is resizable by dragging left edge

    16. Click a skill chip, verify a Claude Code terminal session opens
