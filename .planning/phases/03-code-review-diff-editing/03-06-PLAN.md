- `useAiReview()` - Returns `{ triggerReview, isReviewing }`.

- Uses useMutation from TanStack Query.
- On error, show toast (if sonner is available) or console.error.

Create frontend/src/components/diff/AnnotationGutter.tsx:
- A component rendered as a gutter icon in the @git-diff-view/react diff view
- Shows a small colored icon based on annotation severity:
  - error: AlertCircle (red #ef4444)
  - warning: AlertTriangle (amber #f59e0b)
  - suggestion: Lightbulb (blue #3b82f6)
  - info: Info (gray #9ca3af)
- Icon size: 14px, positioned in the gutter area
- Clickable: opens/toggles the AnnotationPanel and scrolls to that annotation
- Tooltip on hover showing first ~50 chars of the annotation comment

Wire annotations into DiffViewer:

Wire "Review with Claude" button in DiffToolbar:

- When submitted: call useAiReview().triggerReview with the current diff text and optional custom prompt
- While reviewing: show loading spinner in the toolbar button area, disable the button
- When done: annotations appear in diff via extendData, and AnnotationPanel opens automatically
